GM Kit
As an avid D&D player and GM, I like to bring my passion of apps and D&D together to make things that help me run my games. After hooking up a projector to project maps down onto the table that I use when I run my games, I decided to make an app to help with displaying maps.
The app itself takes in any image and allows the user to add/draw walls onto it for dynamic lighting. Over time other small useful features have been added, like local audio track management, random name/info generators, and limited smart bulb support.
More info can be found here: GM Kit
Gravity Well Demo
Interesting little gravity well demo I made a long time ago getting used to the HTML Canvas.
  • Gravity Wells can be placed and set to push or pull asteroids.
  • Gravity Wells can be configured by right clicking on the center of the well.
  • Planets can be placed and will be destroyed by asteroids, creating asteroid like chunks as it gets destroyed.
  • Asteroids are created at the mouse position with an initial direction matching the movement of the mouse.
View it here: Gravity Well
